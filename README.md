# 👻 Ghost Hunt — لعبة الأشباح الفضائية

> لعبة ويب متعددة اللاعبين لحظية مستوحاة من Among Us، مبنية بـ Node.js + Socket.IO

---

## 🎮 طريقة اللعب

### الفئتان
| الفئة | الهدف |
|-------|-------|
| 👨‍🚀 **الطاقم** | أكمل جميع المهام أو اكشف هوية الأشباح بالتصويت |
| 👻 **الأشباح** | تصفية أفراد الطاقم حتى يصبحوا مساوين لهم عدداً |

### سير اللعبة
1. **المضيف** ينشئ غرفة ويحصل على **كود من 5 أحرف**
2. **اللاعبون** يدخلون بنفس الكود من هواتفهم
3. يتم توزيع الأدوار سراً عند بدء اللعبة
4. **الطاقم** يتنقل بين الغرف لإنجاز المهام
5. **الأشباح** يتنقلون ويصفّون الطاقم سراً
6. أي لاعب يجد جثة → يضغط "الإبلاغ عن جثة" → اجتماع + تصويت
7. المصوَّت له يُطرد من اللعبة ويُكشف دوره!

### شروط الفوز
- 🛸 **الطاقم يفوز**: عند إكمال كل المهام أو طرد كل الأشباح
- 👻 **الأشباح يفوزون**: عند مساواة عدد الطاقم الحي

## 🚀 التثبيت والتشغيل محلياً

```bash
git clone https://github.com/YOUR_USERNAME/ghost-hunt.git
cd ghost-hunt
npm install
node server.js
```
افتح المتصفح على: `http://localhost:3000`

---

## ☁️ النشر على Render

1. ارفع الكود على GitHub
2. اذهب إلى [render.com](https://render.com) → **New Web Service**
3. اختر الـ repository
4. Render سيقرأ `render.yaml` تلقائياً
5. اضغط **Deploy** 🎉

> **ملاحظة**: اختر نوع الخادم **Free** للاستخدام المجاني

---

## 🛠️ التقنيات المستخدمة

- **Backend**: Node.js + Express + Socket.IO
- **Frontend**: HTML5 + CSS3 + Vanilla JS
- **Real-time**: WebSockets via Socket.IO
- **Deployment**: Render.com (Web Service)

---

## ⚙️ إعدادات اللعبة

يمكن للمضيف تعديل:
- عدد الأشباح (1-3)
- عدد المهام لكل لاعب (2-5)
- وقت النقاش (20-60 ثانية)
